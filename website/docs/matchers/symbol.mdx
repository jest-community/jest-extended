# Symbol

### .toBeSymbol()

Use `.toBeSymbol` when checking if a value is a `Symbol`.

<TestFile name="toBeSymbol">
  {`test('passes when value is a symbol', () => {
  expect(Symbol()).toBeSymbol();
  expect(true).not.toBeSymbol();
});;`}
</TestFile>
